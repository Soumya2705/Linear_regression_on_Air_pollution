# 🌫️ Air Pollution Analysis using Linear Regression

## 📌 Project Overview

This project analyzes **air pollution trends** using different **Linear Regression techniques** to understand pollution behavior and predict pollution levels.

The dataset is collected from **OpenAQ air quality data**, which contains pollution measurements from multiple locations.

The project demonstrates a complete **machine learning workflow**, including **data preprocessing, data balancing, regression modeling, and model evaluation**.

## ⚖️ Data Balancing

Before training the models, the dataset was balanced to improve model performance and reduce bias.

Techniques used:

* **Oversampling**
* **Undersampling**

Balanced datasets were created to analyze how model performance changes under different data distributions.

## 📈 Models Implemented

### 1️⃣ Linear Regression

A standard linear regression model was used to predict pollution values.

### 2️⃣ Regularized Regression

Regularization techniques were applied to reduce **overfitting** and improve model generalization.

### 3️⃣ Gradient Descent

A **custom gradient descent implementation** was used to optimize regression parameters.
